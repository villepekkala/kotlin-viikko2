# Kotlinperusteet-viikko1
Viikko1 kotitehtävät


'id': Int tunniste, on yksilöllinen
'title' Tehtävän nimi, String
'description' Lyhyt kuvaus, String
'dueDate' Eräpäivä, String
'done' Tehtävän tila, boolean, true = valmis, false = kesken

Addtask, lisää uuden tehtävän listan loppuun.
Done, Kääntää booleanin done tilaan.
filterbyDone, suodattaa listasta näkyviin vain tehtävät, mitkä on suoritettu
sortByDueDate, järjestää listan nousevaan järjestykseen eräpäivän mukaan

HomeScreen, päänäkymä missä on kaikki napit, tekstit yms. Käytetään peruskomponentteja, column, Row, Button jne.
